# NotchHub Installation Guide (macOS)

Follow the steps below to install and open **NotchHub** on your Mac.

## 1. Download the Latest Release

Go to the repository and download the **ZIP file** from the **latest release/deployment**.

- Navigate to the repository's **Releases** section
- Download the most recent `.zip` file for **NotchHub**

## 2. Unzip the Downloaded File

Once the ZIP file has finished downloading:

1. Open your **Downloads** folder (or wherever the ZIP file was saved)
2. Double-click the `.zip` file to unzip it
3. This will create an unzipped **NotchHub** app file/folder

## 3. Move NotchHub to Applications

After unzipping:

1. Open **Finder**
2. Drag the unzipped **NotchHub** app into your **Applications** folder

Once the app has been moved to **Applications**, you can safely delete the original `.zip` file if you no longer need it.

## 4. Open NotchHub

1. Open your **Applications** folder
2. Find and open **NotchHub**

### If macOS Blocks the App

You may see a warning that **NotchHub cannot be opened because it is from an unidentified developer**, or that the app is **not verified/authorized**.

To allow the app to open:

1. Open **System Settings** on your Mac
2. Go to **Privacy & Security**
3. Scroll down until you see a message about **NotchHub** being blocked
4. Click **Open Anyway**
5. Confirm any additional prompts

After this, reopen **NotchHub** from your **Applications** folder and it should launch normally.

---

If you run into any issues, try closing the app completely and reopening it after allowing access in **Privacy & Security**.
